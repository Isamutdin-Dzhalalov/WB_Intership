### Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и порядок их вызовов. 
```
package main
 
import (
    "fmt"
)
 
func test() (x int) {
    defer func() {
        x++
    }()
    x = 1
    return
}
 
 
func anotherTest() int {
    var x int
    defer func() {
        x++
    }()
    x = 1
    return x
}
 
 
func main() {
    fmt.Println(test())
    fmt.Println(anotherTest())
}
```
### Ответ
```
2
1

defer - ключевое слово, которое используется для отложенного выполнения, т.е перед выходом из ф-ции.
В нашем примере мы имеем ф-цию test(), которая имеет именованный возврат переменной, в отличии от простого
возврата в ф-ции anotherTest().
Разница в том, что когда мы имеем ф-цию без именованного возврата(anotherTest()), перед возвратом создаётся копия
возвращаемого значения, и defer срабатывает после того, как создалась копия возвращаемого значения и на возвращамое
значение он не влияет.
Ф-ция с именованным возвратом(test()) возвращает переменную напрямую, без создания копий, соответсвенно изменение
значения в отложенной ф-ции меняет и возвращаемое значение.
```
