### Что выведет программа? Объяснить вывод программы.
```go
package main

func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 10; i++ {
            ch <- i
        }
    }()

    for n := range ch {
        println(n)
    }
}
```

### Ответ
```
Программа выведет значения от 0 до 9(включительно), после чего
выведется ошибка deadlock.
Причина: так как мы не закрываем канал, цикл for range который читает
данные из канала, будет продолжаться, пока канал не будет закрыт, после чего
он автоматически будет закрыт, но канал не закрывается и основная горутина 
заблокируется, ожидая данные, которых не будет.

Чтобы исправить данную ошибку, можно добавить close(ch) закрытие канала,
в ананимную ф-цию после цикла for.
```
